## üì¶ Installation

1. Open VS Code
2. Go to Extensions (`Cmd+Shift+X` / `Ctrl+Shift+X`)
3. Search for "Python AI Engineering Extension Pack"
4. Click Install

Or install via CLI:

```bash
code --install-extension mfmezger.python-ai-engineering
```
